Particle Life Simulation

About the Project

This project simulates the behavior of particles in a controlled environment. It allows users to explore various dynamics of particle interactions and movements.

Controls

You can control the simulation with the following keys:

  • Space: Start/Pause the simulation
  • Arrow Keys: Move particles
  • R: Reset the simulation

Features

  • Real-time particle interaction
  • Customizable parameters
  • Visual effects
