Welcome to Particle Life Simulation

Project Description

This Particle Life Simulation project allows users to explore the life cycle of particles in a dynamic and interactive environment. Through various simulations, you can witness how particles behave based on different parameters.

Controls Information

Use the following controls to interact with the simulation:

  • W - Increase speed
  • S - Decrease speed
  • A - Spawn new particles
  • D - Remove particles
